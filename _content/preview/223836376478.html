골판지 박스 희소식

골판지와 합지하는 박스는 소량 생산이 불가능합니다. 정확하게 말씀드리면 단가와 납기가 맞지 않을 수 있다는 겁니다.

​

이유는 여러 가지가 있지만 골판지 생산의 공급이 소량을 위해 공급하지 않는 것이 보편적인 생산방식이고 최소한 골판지 박스는 500개 또는 수천 개 이상 발주를 해야 단가적으로 맞출 수 있기 때문입니다.

​

또한 샘플용 소량 생산 또한 주요 거래처 아니면 하지 않는 경우가 많습니다.

(주)파파스컴퍼니은 다음과 같은 조건이 맞으면 골판지 합지 박스 샘플을 제작해 드립니다.

​

사이즈 : 46 전지 사이즈 (788mm × 1090mm) 이내

종이 : E 골 골판지 + 아트지 또는 스노우지(180g 또는 200g 이상) 합지

인쇄 : 컬러 4도 (단, UV 평판 인쇄)

코팅 : 유광 또는 무광라미네이팅

후공정 : E 골판지 + 인쇄 합지

후공정 : 샘플용 커팅

샘플 수량 : 5개

​

개당 : 30,000원 이상(상담 후 정확한 견적을 드립니다. 이유는 사이즈와 기타 후공정에 따라 견적이 바뀔 수 있습니다.)

납기 : 상담과 디자인 컨펌 후 2~4일

​

카테고리에 따른 소량 샘플 가능 여부

​

단상자 계열 : 합지 하지 않고 하나의 종이로만 이루어진 박스 형태, 사이즈만 맞으면 가장 저렴하게 소량 생산이 가능합니다.

· 기본 사이즈 : 전개도 A3 이내

· 종이 : 300g ~ 350g 내외 백색 종이

· 인쇄 : 컬러 CMYK 4도

· 코팅 : 유광/무광 선택 가능

· 후공정 : 디지털 커팅

· 기본 수량 : 100개 미만

· 납기 : 2 ~ 5일

골판지 합지 계열 : 골판지와 합지하여 제작되는 택배 박스 형태의 컬러 상자입니다.

· 기본 사이즈 : 46전지 이내

· 종이 : E 골판지 + 합지 종이(아트지, 스노우지 200g 외 다양합니다.)

· 인쇄 : 컬러 UV 인쇄

· 코팅 : 유광/무광 선택 가능

· 후공정 : 디지털 커팅

· 기본 수량 : 10개 미만

· 납기 : 2 ~ 5일

싸바리 합지 계열 : 갱판지와 합지하여 제작되는 표지/2단/3단 형태의 박스입니다.

· 기본 사이즈 : 46 2절 이내

· 종이 : 1200g 또는 1500g 백색 갱판지/흑색 갱판지 + 합지 종이(아트지, 스노우지 150g 외 다양합니다.)

· 인쇄 : 디지털 인쇄

· 코팅 : 유광/무광 선택 가능

· 후공정 : 디지털 커팅, 수동 싸바리

· 기본 수량 : 5개 미만

· 납기 : 2 ~ 5일

다만 납기와 수량 견적은 반드시 상담 후 진행하시기 바랍니다. 박스는 여러 가지 변수가 발생합니다. 시기와 조건에 따라 불가능한 경우도 많습니다. 반드시 전문가와 상담하고 진행하세요.
