2025-04-21 · 공지사항 · 원본

안녕하세요.

소량 생산,

공장직영 맞춤 제작 전문

(주)파파스컴퍼니입니다.

요즘의 트렌드에 맞게

(주)파파스컴퍼니은 소량 다품종

인쇄물에 집중합니다.

물론 대량 주문을 해주시면

너무 감사합니다.

​

하지만 소량에 강점이 있는 만큼

급하신 인쇄물이 필요하시다면

언제든 상담을 요청해 주세요.

(주)파파스컴퍼니에서

소량 생산이 가능한 품목입니다.

​

- 포스터 -

100장 미만 포스터는

당일 생산이 가능합니다.

또한 약 740mm × 510mm

미만이어야 합니다.

​

접수 마감 시간

오전 12시 이전 : 당일 택배 출고

오후 2시 이후 : 익일 택배 출고

​

당일 출고 사이즈

510mm × 740mm 이내

​

해당 수량

100장 이내

단, 추가 수량은 상담 후 결정

​

기타

코팅 : +1일 추가

기타 후공정 : 후공정 당 +1일 추가

​

제약사항

위 사이즈보다 큰 사이즈

별색이나 특별한 색

박인쇄나 기타 후공정이 있는 경우

특별한 종이를 원하시는 경우

디자인까지 요청해 주시는 경우

- 리플렛 -

100장 미만 리플렛도

당일 생산이 가능합니다.

다만 리플렛의 특성상

접지물의 경우 하루 정도

더 소요될 수 있습니다.

​

접수 마감 시간

오전 12시 이전 : 당일 택배 출고

오후 2시 이후 : 익일 택배 출고

​

당일 출고 사이즈

A3, A4, A5

B4, B5

​

해당 수량

100장 이내

단, 추가 수량은 상담 후 결정

​

기타

코팅 : +1일 추가

접지 : 접지 인쇄물은 +1~2일 추가

​

제약사항

종이를 특별한 종이로 선택하는 경우

별색이나 특별한 색상

박인쇄나 기타 후공정이 있는 경우

디자인까지 요청해 주시는 경우

- 스티커 -

50장 자유칼선 스티커는

당일 생산이 가능합니다.

​

접수 마감 시간

오전 12시 이전 : 당일 택배 출고

오후 2시 이후 : 익일 택배 출고

​

당일 출고 사이즈

A3 이내 사이즈

​

해당 수량

50장 이내

단, 추가 수량은 상담 후 결정

​

기타

코팅 : +1일 추가

기타 후공정 : 후공정 당 +1일 추가

​

제약사항

사이즈가 A3를 넘는 경우

특별한 원단을 원하는 경우

칼이 너무 촘촘한 경우

박인쇄나 기타 후공정이 있는 경우

디자인까지 요청해 주시는 경우

- 단상자 -

5개 미만 단상자는

당일 생산이 가능합니다.

​

접수 마감 시간

오전 12시 이전 : 당일 택배 출고

오후 2시 이후 : 익일 택배 출고

​

당일 출고 사이즈

전개도가 A3 이내

​

해당 수량

5장 이내

단, 추가 수량은 상담 후 결정

​

기타

코팅 : +1일 추가

접착 : +1일 추가

기타 옵션 선택 시 상담

예를 들어 더 많은 수량이나

별도 후공정 요청 시

​

제약사항

종이를 특별한 종이로 선택하는 경우

별색이나 특별한 색상

박인쇄나 기타 후공정이 있는 경우

디자인까지 요청해 주시는 경우

기타 인쇄물 출고 시간

​

명함 : 1 ~ 3일

카탈로그 : 3 ~ 5일

실사출력 : 1 ~ 3일

봉투 : 2 ~ 4일

패키지 : 7 ~ 10일

쇼핑백 : 7 ~ 10일

​

디자인 : 페이지당 1 ~ 2일 소요,

총 페이지 7일 이내 1차 시안

(주)파파스컴퍼니은

위의 5가지 품목에 한하여서는

상담을 통해 최대한

빠른 납기를 실현하려고

노력 중에 있습니다.

​

언제든 궁금한 사항이 있으시면

톡톡이나 문의를 주시기 바랍니다.

네이버 톡톡보다는

전화나 카톡 연결이 대응이나

상담이 빠르니 참고하시기 바랍니다.

​
